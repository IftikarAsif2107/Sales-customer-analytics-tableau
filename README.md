# Sales-customer-analytics-tableau
Interactive Tableau dashboard analyzing sales performance, profitability, and customer behavior across regions and categories.

📊 Sales & Customer Analytics Dashboard – Tableau

This project presents an interactive Sales and Customer Dashboard built in Tableau to analyze revenue performance, profitability, and customer behavior across regions and product categories.

The dashboard consolidates key business KPIs such as total sales, total profit, customer count, regional contribution, category performance, and time-based sales trends. It enables dynamic exploration through filters and calculated metrics, allowing users to drill down into performance drivers and identify high-performing segments.

🔎 Key Capabilities

KPI tracking (Sales, Profit, Customer Metrics)

Region-wise and Category-wise performance analysis

Trend analysis over time

Customer contribution insights

Interactive filters and calculated fields

🛠 Tools & Techniques

Tableau Desktop

Calculated fields

Aggregations and KPI cards

Interactive dashboard design

This project demonstrates the ability to transform raw transactional data into actionable business insights using structured visualization and analytical thinking.
